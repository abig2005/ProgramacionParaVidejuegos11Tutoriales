#  Tutoriales 2D en Unity

Este repositorio contiene recursos prácticos para el desarrollo de videojuegos 2D en Unity.  
Aquí encontrarás tanto los **videos de referencia** como la **documentación técnica** que acompañan cada actividad.

## Contenido
- Animación de personajes y creación de controladores.
- Construcción de mapas con Tilemap y Grid.
- Implementación de movimiento y salto en 2D.
- Creación de prefabs y administración de objetos.
- Fondos animados y transición entre niveles.
- Checkpoints con **PlayerPrefs**.

---

##  Cómo usar
1. Descarga o clona este repositorio.  
2. Abre el proyecto en **Unity**.  
3. Revisa los videos y la documentación para seguir cada paso.

## 🛠️ Tecnologías utilizadas
